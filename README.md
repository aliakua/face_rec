## Face Recognition project

**Dataset:** CelebA dataset of 500 faces.

**Classifiers:**
 Angular penalty Models were implemented: 
 * SphereFace,
 * CosFace,
 * ArcFace, 
 * and as *baseline model* was done Softmax_CE.
